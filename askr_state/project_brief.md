# Project Brief

Askr is a CLI-based AI coding agent that manages interactive development sessions with an LLM, persisting state across interruptions and supporting autonomous task continuation when context limits are hit. It solves the problem of losing work context and requiring manual re-entry when long coding sessions exceed token budgets.

## What's In Flight

- Context-cut handover system: When a session hits 75% context limit, Askr automatically creates a checkpoint and launches a new autonomous session to continue the task without user approval.
- Hook integration for pre-compaction blocking: Ensuring the pre_compact.py hook fires before LLM context compaction and prevents auto-compaction when context limits are triggered.
- End-to-end validation: Testing the full context-cut scenario (trigger 75% limit → autonomous task-cut session launches → handover JSON is complete and parseable).
- Handover document structure verification: Confirming the JSON handover format generated by stop.py contains all required fields for autonomous agent continuation.
